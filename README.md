# 🎵 Music Loader

Загрузчик музыки из онлайн-источников с поддержкой интерактивного поиска и пакетной загрузки по плейлисту.

## 📋 Возможности

- 🔍 **Интерактивный поиск** — выбор трека из списка найденных результатов
- 📜 **Пакетная загрузка** — скачивание треков из плейлиста (файл `.txt`)
- ⏱️ **Автоматическая задержка** — пауза между загрузками для избежания блокировок
- 📁 **Гибкое хранение** — настройка папки для сохранения файлов

## 🚀 Требования

- Python 3.13+
- Библиотеки:
  ```bash
  pip install requests bs4 
  ```

## 📖 Использование

### Базовый синтаксис

```bash
python main.py [опции]
```

### Опции

| Опция | Описание |
|-------|----------|
| `-h`, `--help` | Показать справку |
| `-p`, `--path` | Путь для сохранения (по умолчанию: `download/`) |
| `-pl`, `--playlist` | Путь к файлу плейлиста (`.txt`) |

### Примеры

**Интерактивный режим** (поиск и выбор трека):
```bash
python main.py
python main.py -p my_music/
```

**Пакетная загрузка** (из плейлиста):
```bash
python main.py -pl playlist.txt
python main.py -p my_music/ -pl playlist.txt 
```

## 📝 Формат плейлиста

Создайте текстовый файл (например, `playlist.txt`) в формате:
```
артист название_трека
еще артист еще трек
```

Каждая строка — отдельный запрос на поиск и загрузку.


## ⏱️ Задержка между загрузками

При использовании плейлиста между скачиваниями треков автоматически добавляется задержка в **3 секунды** для предотвращения временной блокировки со стороны сайта.
